# Emma Code Minecraft Hosting Website

A static, whimsical cottage‑themed Minecraft hosting website built with HTML5, CSS3, and vanilla JavaScript.

## Features
- Responsive design with mobile navigation
- Accessible FAQ accordion and modal popups
- Contact form with client‑side validation
- Back‑to‑top button
- Semantic HTML5 and BEM CSS
- Modular JavaScript (IIFE pattern)
- WCAG AA compliant color contrast
- Prefers‑reduced‑motion support

## Build
No build tools required. Serve the files with any static web server.
