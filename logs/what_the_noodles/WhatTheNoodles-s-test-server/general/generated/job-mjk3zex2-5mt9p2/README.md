# AetherHost - Premium Minecraft Hosting UI

This is a high-end, dark-themed multi-page website for a Minecraft hosting provider, featuring a functional mock Pterodactyl console.

## Features
- **Interactive Console**: Real-time command simulation with security filters.
- **Glassmorphism**: Modern UI using backdrop filters and deep shadows.
- **Responsive**: Fully optimized for mobile and desktop.
- **Multi-page**: Includes Home, Plans, and About pages.

## Setup
1. Extract all files maintaining the folder structure.
2. Open `index.html` in any modern web browser.
3. To test the console, try typing `/spark tps` or `/list`.
