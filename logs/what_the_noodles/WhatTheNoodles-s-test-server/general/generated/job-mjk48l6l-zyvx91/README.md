# AetherNode Minecraft Hosting Template

An elite, dark-themed multi-page website for Minecraft hosting providers.

## Features
- **Live Mock Console**: Interactive terminal on the homepage with command filtering (allows /spark, /list, etc., blocks /stop, /op).
- **Glassmorphism UI**: Modern frosted glass effects and deep shadows.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Multi-page**: Includes Home, Plans, and About pages with consistent navigation.
- **Animations**: Intersection Observer for scroll-reveal effects and CSS keyframes for pulses.

## Setup
1. Extract all files.
2. Open `index.html` in any modern web browser.
3. To customize the console logic, edit `js/main.js`.
