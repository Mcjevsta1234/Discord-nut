# Emma Code Cute Cottage Minecraft Hosting Site

A whimsical, static landing page for a Minecraft server hosting service.

## Features
- Responsive design with pastel cottage aesthetics
- Hero section with animated title
- Three hosting plans with interactive cards
- Accessible modal signup form
- FAQ accordion
- Blog section with local JSON data
- Contact form with toast feedback
- Mobile hamburger menu
- Progressive enhancement (works without JS)

## Tech Stack
- HTML5, CSS3, vanilla JavaScript ES6
- Google Fonts (Lobster, Open Sans)
- Font Awesome 5 Free icons
- No frameworks or build tools

## Color Palette
- Primary Mint: #B6EADA
- Secondary Sky: #AEDFF7
- Accent Forest: #5C8B5B
- Background Cream: #FAF4E4
- Text Dark: #3A3A3A
- Focus Ring: #FF4C4C

## Typography
- Headers: "Lobster", cursive
- Body: "Open Sans", sans-serif
- Base: 1rem

## Deployment
1. Upload the entire folder to any static hosting (GitHub Pages, Netlify, Vercel, etc.)
2. Ensure assets/blog/posts.json is accessible relative to index.html

## Accessibility
- ARIA roles and attributes
- Keyboard navigation and focus trapping
- Skip navigation link
- High contrast colors (WCAG AA)
- Alt texts for all images

## Browser Support
- Chrome, Firefox, Safari, Edge (latest versions)
