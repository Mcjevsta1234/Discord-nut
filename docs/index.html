Discord-nut

Discord + OpenRouter TypeScript LLM Router

A streamlined Discord bot with OpenRouter-backed chat, tool execution, image generation, and a console mode. This page condenses everything you need to set up, run, and operate the bot.

Smart triggers Tool planning Model tiers Console mode Cost awareness Image generation ready

Quick Start

Setup 
npm install
cp .env.example .env
# fill DISCORD_TOKEN, DISCORD_CLIENT_ID, OPENROUTER_API_KEY
npm run dev   # hot reload
npm run build && npm start  # production

Node.js: 16+ recommended. The bot responds when mentioned, replied to, or when its name appears.

Run Modes

Discord & Console
  • Discord: npm run dev (or npm start after build)
  • Console-only: npm run console or npm start -- --console
  • Hybrid (Discord + Console): npm start -- --hybrid

Commands (console): /help, /personas, /persona <name>, /clear, /exit.

Key Features

Behavior
  • LLM routing across tiers (INSTANT/SMART/THINKING/CODING)
  • Action planning + tool execution (MCP tools included)
  • Conversation memory with summarization
  • Image generation hooks preserved (OpenRouter models)
  • Cost tracking per message when enabled

Environment

.env
  • DISCORD_TOKEN, DISCORD_CLIENT_ID (required for Discord)
  • OPENROUTER_API_KEY (required)
  • MODEL_INSTANT, MODEL_SMART, MODEL_THINKING, MODEL_CODING
  • MODEL_*_INPUT_PRICE, MODEL_*_OUTPUT_PRICE for cost tracking
  • CONSOLE_MODE or HYBRID_MODE to change launch mode
  • IMAGE_MODEL, IMAGE_DEFAULT_WIDTH, IMAGE_DEFAULT_HEIGHT for image generation

Commands & Triggers

Discord
  • Responds to mentions, replies, or name mentions in-channel.
  • Personas: set via prompt cues; console mode offers explicit /persona.
  • Memory: keeps recent turns; summarizes older context to stay within token budget.

Troubleshooting

Tips
  • Bot silent? Check intents & permissions; ensure required env vars are set.
  • OpenRouter 401? Verify API key and quota.
  • Build errors? Remove dist, reinstall deps, run npm run build.
  • Image failures? Confirm IMAGE_MODEL exists on OpenRouter.
